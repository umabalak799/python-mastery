4. Program to reverse a number.

num = int(input("Enter the number:"))
rev = 0

while num > 0:
    digit = num % 10
    rev = rev * 10 + digit
    num = num // 10
print ("Reversed num :", rev)

10. List to print count of even and odd numbers

num = list(map(int, input("Enter the numbers: ").split()))

even = 0
odd = 0

for n in num:
    if n % 2 == 0:
        even += 1
    else:
        odd += 1

print("Even numbers: ", even)
print("Odd numbers: ", odd)

11. Remove duplicates from a list

num = list(map(int, input("Enter the numbers: ").split()))
unique = []

for i in num:
    if i not in unique:
        unique.append(i)

print("Unique numbers: ", unique)
